### Стъпка 1: Стартирайте сървъра (ако не е стартиран)
```powershell
cd C:\Users\dbsupport2\PycharmProjects\equipmentsystem
python manage.py runserver
```

### Стъпка 2: Отворете браузър
```
http://127.0.0.1:8000/equipment/
```

### Стъпка 3: Отворете Equipment Detail
- Кликнете на "👁️" бутона за някое оборудване
- Scroll down - ще видите:
  - 🟢 **Тагове** секция (зелен header)
  - ⚫ **Документи** секция (сив header)

### Стъпка 4: Добавете тагове/документи (ако няма)
```
http://127.0.0.1:8000/admin/
```
1. Login като admin
2. Equipment → Equipment
3. Изберете оборудване
4. Scroll до "Tags" и "Documents"
5. Използвайте filter_horizontal widget
6. Save
7. Refresh Equipment Detail страницата

## 📊 Какво точно беше направено?

### Backend (equipment/views.py)
```python
# 1. Добавен import
from .models import Tag, Document

# 2. EquipmentDetailView - добавени tags и documents в context
context['tags'] = equipment.tags.all()
context['documents'] = equipment.documents.all()

# 3. EquipmentListView - добавен prefetch и tag filtering
queryset = Equipment.objects.prefetch_related('tags').all()
if tag_id:
    queryset = queryset.filter(tags__id=tag_id)

# 4. EquipmentListView - добавени tags в context за filter
context['tags'] = Tag.objects.filter(is_active=True).annotate(
    equipment_count=Count('equipment')
)
```
