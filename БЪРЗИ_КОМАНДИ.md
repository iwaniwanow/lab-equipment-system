## Бързи примери с Tag и Document

### Пример 1: Създаване на таг
```powershell
python manage.py shell
```
```python
from equipment.models import Tag

tag = Tag.objects.create(
    name='Калибровано', 
    color='#00CC00',
    description='Калибрирано оборудване'
)
print(f"Създаден таг: {tag}")
```

### Пример 2: Добавяне на таг към оборудване
```python
from equipment.models import Tag, Equipment

tag = Tag.objects.get(name='Калибровано')
equipment = Equipment.objects.first()
equipment.tags.add(tag)
print(f"Тагове на {equipment}: {equipment.tags.all()}")
```

### Пример 3: Търсене на оборудване по таг
```python
from equipment.models import Tag

tag = Tag.objects.get(name='Критично')
equipment_list = tag.equipment.all()
print(f"Оборудване с таг '{tag.name}':")
for eq in equipment_list:
    print(f"  - {eq}")
```

### Пример 4: Създаване на документ
```python
from equipment.models import Document
from django.contrib.auth.models import User

user = User.objects.first()
doc = Document.objects.create(
    name='Валидационен протокол',
    document_type='validation',
    description='Валидация на pH метър',
    uploaded_by=user
)
print(f"Създаден документ: {doc}")
```

### Пример 5: Свързване на документ с оборудване
```python
from equipment.models import Document, Equipment

doc = Document.objects.first()
equipment = Equipment.objects.first()
doc.equipment.add(equipment)
print(f"Документи на {equipment}:")
for d in equipment.documents.all():
    print(f"  - {d}")
```

## Performance testing

### Проверка на query count
```python
from django.db import connection
from django.test.utils import override_settings
from equipment.models import Equipment

# Enable query logging
with override_settings(DEBUG=True):
    equipment_list = Equipment.objects.prefetch_related('tags', 'documents').all()
    for eq in equipment_list:
        print(f"{eq}: {eq.tags.count()} tags, {eq.documents.count()} docs")
    print(f"\nTotal queries: {len(connection.queries)}")
```
