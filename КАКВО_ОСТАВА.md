## 🚀 Бърз старт за използване СЕГА

### Вариант 1: Django Admin Panel
```
1. Отворете: http://127.0.0.1:8000/admin/
2. Login
3. Отидете на Equipment → Tags
4. Създайте нови тагове
5. Използвайте filter_horizontal за добавяне на оборудване
```

### Вариант 2: Python Shell
```python
python manage.py shell

from equipment.models import Tag, Equipment

# Създаване на таг
tag = Tag.objects.create(name='Калибровано', color='#00CC00')

# Добавяне към оборудване
equipment = Equipment.objects.get(asset_number='HS-014')
equipment.tags.add(tag)

# Виждане на всички тагове
print(equipment.tags.all())
```
