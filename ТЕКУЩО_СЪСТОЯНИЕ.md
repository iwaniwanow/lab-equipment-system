## 🎯 Какво може да се направи СЕГА

### 1. Използване на Tag system
```python
from equipment.models import Tag, Equipment

# Създаване на нов таг
tag = Tag.objects.create(
    name='Калибровано',
    color='#00CC00',
    description='Калибрирано оборудване'
)

# Добавяне на таг към оборудване
equipment = Equipment.objects.get(asset_number='HS-014')
equipment.tags.add(tag)

# Намиране на всичко оборудване с даден таг
critical_equipment = Tag.objects.get(name='Критично').equipment.all()
```

### 2. Използване на Document system
```python
from equipment.models import Document, Equipment

# Качване на документ
doc = Document.objects.create(
    name='Валидационен протокол',
    document_type='validation',
    file='path/to/protocol.pdf',
    uploaded_by=request.user,
    version='1.0'
)

# Свързване с оборудване
equipment = Equipment.objects.get(asset_number='WS-0001')
doc.equipment.add(equipment)

# Виждане на всички документи за оборудване
equipment.documents.all()
```

### 3. Работа в Admin Panel
1. Отворете: http://127.0.0.1:8000/admin/
2. Login с вашия admin акаунт
3. Навигирайте до **Equipment** → **Tags**
4. Click **Add Tag** за нов таг
5. Използвайте filter_horizontal widget за лесно добавяне на оборудване

## 🎓 Как да работите с M2M relationships

### Добавяне
```python
# От страната на Equipment
equipment.tags.add(tag1, tag2, tag3)
equipment.documents.add(doc1, doc2)

# От страната на Tag/Document
tag.equipment.add(eq1, eq2, eq3)
doc.equipment.add(eq1, eq2)
```

### Премахване
```python
equipment.tags.remove(tag1)
equipment.documents.remove(doc1)

tag.equipment.remove(eq1)
doc.equipment.remove(eq1)
```

### Изчистване на всички
```python
equipment.tags.clear()
equipment.documents.clear()

tag.equipment.clear()
doc.equipment.clear()
```

### Query
```python
# Всички тагове за оборудване
equipment.tags.all()

# Всички документи за оборудване
equipment.documents.all()

# Всичко оборудване с даден таг
tag.equipment.all()

# Всичко оборудване с даден документ
doc.equipment.all()
```

### Filtering
```python
# Оборудване с конкретен таг
Equipment.objects.filter(tags__name='Критично')

# Оборудване с документи от тип 'manual'
Equipment.objects.filter(documents__document_type='manual')

# Оборудване БЕЗ тагове
Equipment.objects.filter(tags__isnull=True)
```

## 💡 Забележки

1. **Encoding issue**: Някои тагове имат криво кодиране на кирилицата в конзолата, но в базата данни и admin panel всичко е наред.

2. **File uploads**: За Document.file полето трябва да се конфигурира MEDIA_ROOT в settings.py.

3. **Permissions**: DocumentAdmin автоматично записва uploaded_by като текущия потребител.

4. **Performance**: При много оборудване, използвайте `.prefetch_related('tags', 'documents')` за по-бързи queries.
