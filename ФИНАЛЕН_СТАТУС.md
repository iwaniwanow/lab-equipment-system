## 🎯 Примерен код

### Работа с тагове
```python
from equipment.models import Tag, Equipment

# Създаване на таг
tag = Tag.objects.create(
    name='Калибровано',
    color='#00CC00'
)

# Добавяне към оборудване
equipment = Equipment.objects.get(asset_number='HS-014')
equipment.tags.add(tag)

# Виждане на всички тагове
print(equipment.tags.all())
```

### Работа с документи
```python
from equipment.models import Document, Equipment

# Създаване на документ
doc = Document.objects.create(
    name='Валидационен протокол',
    document_type='validation',
    uploaded_by=request.user
)

# Свързване с оборудване
equipment = Equipment.objects.get(asset_number='WS-0001')
doc.equipment.add(equipment)

# Виждане на документите
print(equipment.documents.all())
```
